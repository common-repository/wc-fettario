Italian feature add-on for Woocommerce è un semplice plugin che permette di rendere Woocommerce in linea con il regime forfettario

== Description ==
=== RENDI WOOCOMMERCE IN LINEA CON IL REGIME FORFETTARIO ===
Italian feature add-on for Woocommerce ti permette di inserire nella generazione degli ordini la marca da bollo e, se impostata, la rivalsa contributiva

== Installation ==
= Installazione automatica =

Per installare automaticamente Italian feature add-on for Woocommerce ti basterà andare sulal sezione plugin di Wordpress cercare tra la reposity Italian feature add-on for Woocommerce\" e cliccare su installa e successivamente su attiva

= Installazione manuale =

Scaricare sulla reposity di Wordpress il plugin Italian feature add-on for Woocommerce e procedere con la funzione carica plugin sulla scheda \"Aggiungi nuovo\" della sezione plugin il file .zip scaricato.

= Aggiornamenti =

Gli aggiornamenti saranno visibili ed installabili automaticamente
